# Prompts

AI-executable prompt definitions for automated tasks. Each prompt lives in its own subdirectory with a `README.md` entry point.

Prompts use `<execution>` tags for sequential steps and `<rules>` tags for constraints. For the full authoring conventions, see [ai-authoring.md](../foundation/ai-authoring.md) (Prompt-Specific Patterns section).

## Available Prompts

| Prompt                                | Purpose                                              |
| ------------------------------------- | ---------------------------------------------------- |
| [save](save/README.md)               | Stage all changes and generate a conventional commit |
| [docs-audit](docs-audit/README.md)   | Audit documentation for compliance and redundancy    |
